Pull Requests (PRs)

A Pull Request is a formal way to say: "Hey, I made some changes — please review and merge them."

What Is a Pull Request?

Real-World Analogy

📋 Think of a PR like submitting a report to your manager. You've done the work, now you're asking them to review it and approve it before it goes into the official project. They might ask for changes before approving.
