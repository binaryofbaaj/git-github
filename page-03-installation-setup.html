Installing Git & Setting Up

Let's get Git on your computer and configure it. Don't worry — the terminal is less scary than it looks.

First: What's a "Terminal"?

Before we install Git, let's demystify the thing you'll be typing Git commands into — the terminalTerminalA text-based interface to give your computer instructions by typing commands instead of clicking buttons. Also called "command prompt" (Windows) or "shell"..

Real-World Analogy

A terminal is just a text-based way to give your computer instructions. Instead of clicking icons and buttons, you type short commands. It's like texting your computer instead of having a face-to-face conversation. Same result, different method.

Don't be intimidated by the terminal! You only need to learn about 10-15 commands total, and we'll teach them one at a time. Think of it like learning a few magic words — each one does something useful.

Installing Git

Choose your operating system below to see the right installation steps:

Option 1: Using Homebrew (recommended)

If you have Homebrew installed, just run:

Terminal
brew install git

Option 2: Xcode Command Line Tools

Open Terminal and type:

Terminal
xcode-select --install

A popup will appear asking to install developer tools. Click "Install" and wait.

Download Git for Windows

  1. Go to git-scm.com/download/win
  2. Download the installer (.exe file)
  3. Run the installer — the default settings are fine for beginners
  4. When asked about a text editor, you can leave the default or choose VS Code
  5. Click "Finish" when done

After installing, open Git Bash (a terminal that comes with Git for Windows) from the Start menu.

Windows users: use Git Bash (installed with Git) instead of Command Prompt. Git Bash understands the same commands as Mac/Linux terminals.

Debian / Ubuntu

Terminal
sudo apt update
sudo apt install git

Fedora

Terminal
sudo dnf install git

Verify the Installation

No matter what OS you're on, open a terminal and run:

Terminal
git --version

If you see something like git version 2.42.0, you're good to go! 🎉

Think of git config as filling out your contact card. Git stores it in its address book so it knows who to credit for each change.

Key Takeaways

🎉 You've installed Git and configured your identity!

  • The terminal is just a text-based way to talk to your computer
  • Git installs easily on Mac, Windows, and Linux
  • git config stores your name and email for commits
  • git --version confirms Git is installed correctly
