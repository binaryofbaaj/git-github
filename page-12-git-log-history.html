Viewing History: git log, diff & blame

Explore your project's past — see every change, compare versions, and find out who wrote what.

git log — See the Timeline

git log shows the full history of commits — like scrolling through a timeline of every save point.

Terminal — Exploring History
$

git diff — See What Changed

git diff shows the exact changes between versions — lines added in green and removed in red.

Terminal
# See unstaged changes
git diff

# See staged changes (ready to commit)
git diff --staged

# Compare two commits
git diff a1b2c3d 7e8f9g0

git blame — Who Wrote This?

git blame shows who last edited each line of a file. Despite the name, it's not about blaming anyone — it's about finding context!

😄 "git blame — it sounds harsh", but it's just a tool for finding who to ask for context, not who to yell at. It's incredibly useful when you need to understand why a line was written a certain way.

🎉 You can now explore Git history! Use git log to see commits, git diff to see changes, and git blame to find authors.
